# PROBE 2026 Workshop Website

[![Built with Starlight](https://astro.badg.es/v2/built-with-starlight/tiny.svg)](https://starlight.astro.build)

Source code for the PROBE 2026 workshop website, built with Astro and Starlight. The content in src/content/docs defines the site pages and challenge descriptions, while configuration and UI components live under src/ and astro.config.mjs.
